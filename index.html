FastMaxFlow

Nearly maximum flows in nearly linear time and other flow approximation tools

Download Download View FastMaxFlow on GitHub

What?

A python implementation of an algorithm for solving maximum flow (or more generally, minimum congestion) problems on a capacitated graph as described by Jonah Sherman [1].

The resulting flow assignment is epsilon-accurate and takes time "nearly" proportional to the number of edges, and inversely proportional to the square of the error term epsilon.

Additionally, implementations of the Nagamochi-Ibaraki edge connectivity certificate [2, 3] and Benczur and Karger's approximate cut preserving sparsification [4] are provided.

Citations

  1. J. Sherman: Nearly Maximum Flows in Nearly Linear Time, arXiv:1304.2077.
  2. H. Nagamochi, T. Ibaraki: Computing the edge-connectivity of multigraphs and capacitated graphs, SIAM J. Discrete Mathematics vol.5, 1992, pp.54-66.
  3. H. Nagamochi, T. Ibaraki: A linear-time algorithm for finding a sparse k-connected spanning subgraph of a k-connected graph, Algorithmica vol.7, 1992, pp.583-596.
  4. A. Benczur, D. Karger: Randomized Approximation Schemes for Cuts and Flows in Capacitated Graphs, arXiv:cs/0207078.
